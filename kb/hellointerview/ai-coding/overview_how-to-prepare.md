Update after AI-enabled interview :
In my case, they make it clear that you should understand and discuss a solution before hand. Essentially, I used these steps and I think it was effective. My AI got the right answer first time and they seemed happy with the answers:

Think out loud how you will update code or fix bug.
Specify new time and space complexity for the proposed solution
Confirm solution with interview
Ask AI to implement the specific solution
Read through the AI's proposed code solution and make any adjustments that weren't part of your plan
Confirm with interviewer and ask AI to implement the code
Review changes in the code again and approve only the ones that make sense. Sometimes the AI will start making unnecessary changes.
Hope test runs successfully and repeat
