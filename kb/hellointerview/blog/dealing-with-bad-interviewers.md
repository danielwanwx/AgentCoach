Oye, this is tough. If they're visibly distracted you might try asking them a few deeper questions. Interviewers do this (a) because they have a lot of pressure for other commitments, (b) because they aren't invested in the process and are doing interviews because they're obligated to. In either case they're under the illusion that you're not noticing (this is obviously stupid, but we tell ourselves all kinds of stories).

If you can shatter that illusion by forcing them to come to grips with "wow, I don't know the answer to this ... I better pay attention", you might be able to pull them back in. A lot of candidates tend to ask closed-ended, clarifying questions in the interview. It can be startling (or positively: refreshing) when a candidate goes a bit off script. So rock the boat a little and see if you can get them to wake up.
